**Project Description:**<br>
Based on the tables given below, cube and queries have been solved:
Tb_Supplier(Supp_ID, Name, City, State)<br>
Tb_Consumer(Con_ID, Name, City, State)<br>
Tb_Product(Prod_ID, Name, Product_Category, Product_Line, Product_Packaging)<br>
Tb_Offers(Supp_ID, Prod_ID, Quantity, Price)<br>
Tb_Requests(Con_ID, Prod_ID, Quantity, Price)<br>
Tb_Transactions(Tran_ID, Supp_ID, Con_ID, Prod_ID, Quantity, Price)<br>
Tb_Date(DateKey, DateYear, DateMonth, DateMonthName, DateWeek,
DateDayOfMonth, DateDayOfWeek, FiscalWeek, IsWeekEnd, IsHoliday, HolidayName)<br>
Tb_Time(TimeKey, TheHour, IsLunchHour, IsBusinessHour, ShiftNumber)<br>
